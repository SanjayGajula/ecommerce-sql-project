# E-Commerce Sales Database (SQL Project)

## Description
This project demonstrates SQL skills through the design and analysis
of an E-Commerce Sales Database.

## Database
MySQL

## Features
- Customer & Order Management
- Sales Analytics
- Revenue & Ranking Reports

## SQL Concepts Used
- Joins
- Group By & Having
- Subqueries
- Window Functions

## Use Cases
- Top-selling products
- Monthly revenue analysis
- Customer ranking by spending
